Développeur Web

Le développeur web est indispensable dans une équipe de développement. L'informatique n'a aucun secret pour lui et il a pour mission de donner vie aux maquettes visuelles d'un site ou d’une application web. Le développeur web doit traduire, en langage web, ses recommandations graphiques en rédigeant du code HTML, CSS, mais aussi JavaScript. Sommaire
